Music Assistant RPM repository

Add the Music Assistant RPM repository:

sudo tee /etc/yum.repos.d/music-assistant.repo <<'REPO'
[music-assistant]
name=Music Assistant
baseurl=https://music-assistant.github.io/desktop-app/rpm/$basearch
enabled=1
gpgcheck=1
repo_gpgcheck=1
gpgkey=https://music-assistant.github.io/desktop-app/rpm/RPM-GPG-KEY-music-assistant
REPO
sudo dnf install music-assistant

After setup, update with sudo dnf upgrade music-assistant.
